## 📋 Installation Instructions

### Step 1: Install VSCode
Download and install VSCode from: https://code.visualstudio.com/

### Step 2: Install NLight Extension
1. Open VSCode
2. Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac)
3. Type: `Extensions: Install from VSIX...`
4. Select the downloaded `nlight-2.2.2.vsix` file

### Step 3: Setup AI (One-time)
1. In VSCode, press `Ctrl+Shift+P`
2. Type: `NLight: Setup AI (One-time)`
3. Click Enter
4. Wait for the setup to complete (downloads AI model)

### Step 4: Start Using NLight
1. Look for **NLight** in the left sidebar (Activity Bar)
2. Click on **Chat**
3. Start chatting with your AI!

## 🔧 Troubleshooting

### If setup fails:
1. Restart VSCode
2. Try the setup command again

### If AI doesn't respond:
1. Check if setup completed successfully
2. Restart VSCode
3. Open NLight chat and try again
